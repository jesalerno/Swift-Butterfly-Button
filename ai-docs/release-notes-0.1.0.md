# ButterflyButton 0.1.0 Release Notes

## Highlights
- New SwiftUI-first `ButterflyButton` control package.
- Deterministic spin interaction with interrupt-on-new-input behavior.
- Built-in validation and safe fallback behavior.
- Accessibility and localization defaults included.
- Unit tests and CI automation included.

## Known Limitations
- UIKit/AppKit wrapper adapters are not included in 0.1.0.
- Advanced UI automation tests are pending expansion in future iterations.
