# Changelog

## 0.1.0 - 2026-02-14
- Initial Swift Package for `ButterflyButton`.
- SwiftUI-first universal control for iOS/iPadOS/macOS.
- Added rendering primitives for mount, axle, medallion, and label layout.
- Added deterministic interaction logic with interrupt policy.
- Added reduced-motion fallback animation mode.
- Added localization resources and accessibility semantics.
- Added Swift Testing unit suite and CI workflow.
- Added vulnerability scan script and report artifact path.
